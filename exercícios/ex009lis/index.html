Trabalhando com listas

Listas ordenadas

São todas aquelas listas onde a ordem dos itens é algo muito importante. Para criarmos uma ordered lists, vamos usar a tag <ol> para delimitar a lista e <li> (list item) para identificar cada item da lista.

  1. Acordar
  2. Arrumar a cama
  3. Fazer café da manhã
  4. Comer
  5. Escovar os dentes
  6. Ir estudar

Listas não ordenadas

São chamadas de unordered list, também chamadas de listas com marcadores.Neste tipo de lista, a ordem dos itens não influencia no seu significado. ela é apenas uma forma de organizar os itens que não apresentam uma classificação necessariamente.

Para criar uma unordered list,usamos a tag <ul> para delimitar a lista e a tag <li> para criar cada um dos itens internos.

  1. açúcar
  2. feijão
  3. óleo
  4. tomate
  5. arroz
  6. sabão em pó

listas mistas

  1. Capítulo 1
  2. Capítulo 2
    1. Item 1
    2. Item 2
    3. Item 3
  3. Capítulo 3
  4. Capítulo 4

Listas de definições

È uma lista sem demarcadores, mas bem útil em alguns casos.

Toda lista de definições está dentro de uma tag <dl> (definition list). Cada termo é um <dt> (definition therme). Cada descrição é um <dd> (definition description).
